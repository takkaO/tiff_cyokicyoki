〇 ソフトウェア概要

TIFF画像を一括でトリミングするソフトウェアです。  

〇 基本的な使用方法

0. ソフトを起動し、処理するTIFF画像があるフォルダを指定する
1. トリミングする領域を指定する
2. Enterキーを押す

トリミング後の画像は、最初に指定したフォルダ内の crop_result フォルダに保存されます。
トリミング元のファイルは変更されません。
